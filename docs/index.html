Is Icicle Right For Your Team?

Icicle helps data science and business intelligence teams collaborate and work more efficiently by making making their feature engineering and business intelligence queries sharable, transparent and fast.

Icicle unique features allow business to share blazingly fast batch queries and run them in streaming mode without any modifications.

Using static type checking with a simple but novel modal type system, we can be 10x faster than Spark, allow different users' queries to be efficiently fused, and guarantee that when a query type checks, it won't crash or interfere with any other user's.
